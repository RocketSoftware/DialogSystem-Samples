Dialog System applications and HTML

This HTML Page is embedded in a normal window within a Dialog System GUI Application.

You may need to configure the HTML Page source, so the images below are loaded from the correct demo directory local to your current runtime environment.

This demonstration illustrates how - when using Microsoft Internet Explorer 5 - you can use the Webbrowser ActiveX to embed HTML pages into your application to vastly improve presentation, with the important ability to use the HTML elements within your application context.
